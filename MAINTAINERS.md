# Maintainer notes

## Submitting to Tinted Theming

1. Fork and clone [`tinted-theming/schemes`](https://github.com/tinted-theming/schemes).
2. Copy both files from this project's [`base16/`](base16/) directory into the fork's `base16/` directory.
3. Run the checks used by the Tinted Theming repository.
4. Open a pull request against its `spec-0.11` branch.

Tinted Theming accepts scheme YAML files directly in its central schemes repository. This project remains the development home for the Arroz con Dulce pair.
