# Arroz con Dulce Tinted Theme

Arroz con Dulce Tinted Theme is a pair of Base16 colour schemes inspired by Puerto Rican arroz con dulce: coconut cream and rice, cinnamon and clove, toasted sugar, and raisins.

- `arroz-con-dulce.yaml` is the warm light variant.
- `arroz-con-dulce-dark.yaml` is the cocoa-and-clove dark variant.

The canonical scheme sources are in [`base16/`](base16/). They follow the Tinted Theming 0.11 YAML format and are ready to be added, unchanged, to the [`base16/` directory of `tinted-theming/schemes`](https://github.com/tinted-theming/schemes/tree/spec-0.11/base16).

## Submitting to Tinted Theming

1. Fork and clone [`tinted-theming/schemes`](https://github.com/tinted-theming/schemes).
2. Copy both files from this project's `base16/` directory into the fork's `base16/` directory.
3. Run the checks used by the Tinted Theming repository.
4. Open a pull request against its `spec-0.11` branch.

Tinted Theming accepts scheme YAML files directly in its central schemes repository; this project remains the development home for the Arroz con Dulce pair.
